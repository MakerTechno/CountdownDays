# 🎺Countdown Days —— Our day, our treasured time.🎼
Welcome! This is a repository for an AI-cooperative Swing UI project organized by MakerTechno.

Wish we could leave both past and future behind, and hold close the very moment we share now.

## 🎻How to use>>>
This project was created with **Java 21, Windows10-22H2 system**.

I haven't tested it with other environments yet, so we can't promise its compatibility with untested ones.

If you've installed the Microsoft build of Java 21, then the only thing you need to do is launch this jar.

Otherwise, you may install OpenJDK 21 and run `java -jar absolute-path-of-your-jar`.

To **change settings, switch languages, or enter window mode**, simply **right-click** on the **text panel** to summon the menu.

To **exit window mode**, **double-click the tray icon**, and it will escape the background WorkerW layer.

Yes, it's not just a simple text component... I admit I’ve poured a bit more passion into it, hoping to make it better.
