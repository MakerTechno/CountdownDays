## 🎻 如何使用 >>>
本项目使用 **Java 21、Windows 10 22H2 系统** 创建。

我没在其他环境中测试，所以无法保证它在什么杂七杂八的环境中的功能。

如果你装了 Microsoft buid Java 21，那双击jar选择java打开就能运行。

或者去装个 OpenJDK 21，然后运行这段代码也行：`java -jar 你的JAR文件的绝对路径`

要 **更改设置、切换语言或进入桌面模式**，只需在 **显示文本** 上 **右键单击** 即可调出菜单。

要 **退出窗口模式**，请 **双击托盘图标**，界面会自己跳出后台WorkerW图层。
